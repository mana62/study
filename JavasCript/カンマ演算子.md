# カンマ演算子
＝カンマ演算子（,） は、「複数の式を左から順に評価し、最後の式の値だけを返す」

## カンマ演算子とは
```js
(式1, 式2, 式3)
```
- 左から順に評価される
- 返ってくる値は一番右の式の結果だけ

## 例
### ① 単純な例
```js
let x = (1, 2, 3);
console.log(x); // 3
```
1 → 評価されるが捨てられる
2 → 評価されるが捨てられる
3 → これが返る

### ② 副作用を伴う処理で使われることがある
```js
let a = 0;
let b = (a = 5, a + 10);

console.log(b); // 15
```
a = 5 が実行される
a + 10 が評価され、これが b に入る

### ③ for 文の更新部分でたまに使われる
```js
for (let i = 0, j = 10; i < j; i++, j--) {
  console.log(i, j);
}
```

```js
// 実際の出力
0 10
1 9
2 8
3 7
4 6
```
i++ と j-- を同時に書けるのはカンマ演算子のおかげ

## ⚠️ 注意
- 普段は使わない方がいい
- 可読性が落ちる
- ただし for 文の更新部など、限定的な場面では便利

## まとめ
- 「カンマ演算子は、複数の式を順に実行し、最後の式の値だけ返す演算子」
