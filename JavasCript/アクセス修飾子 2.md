# アクセス修飾子
＝JSでのアクセス修飾子は`public`と`private`のみ！！！2種類
＝`private`**クラス**のみで使用
＝クラス内でprivateで扱いたい場合は、プロパティの前に **#** をつけるだけ

```js
// JavaScriptのアクセス修飾子

class Pokemon {
// privateはここで先に定義した方がいい
#hp;
  // public（デフォルト）- どこからでもアクセス可能
  name = "ピカチュウ";

  attack() {
    console.log("攻撃！");
  }

  // private（#をつける）- クラス内部のみアクセス可能
  #hp = 100;

  #heal() {
    this.#hp += 10;
  }
}```

⚠️ クラス外で#を使うとエラー

## まとめ
✅ public - デフォルト、どこからでもアクセス可能
✅ private（#） - クラス内部のみ、クラス専用
