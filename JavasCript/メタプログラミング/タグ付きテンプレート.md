# タグ付きテンプレート
- **テンプレートリテラル（バッククォート ``）に関数をくっつけて、文字列の中身を自由に処理できる機能**
- 普通のテンプレートリテラルは文字列をそのまま作るだけですが、
タグ付きテンプレート は **「テンプレート文字列の中身を関数で処理して結果を返す」**ことができる
- 👉 つまり **文字列をプログラムで自由に変換・加工できる → メタプログラミングっぽい**

普通のテンプレートリテラル
```js
const name = 'アクア';
console.log(`こんにちは、${name}です！`);
// → こんにちは、アクアです！
```

⬇️ タグ付きテンプレートにすると
```js
function shout(strings, ...values) {
  return strings[0] + values[0].toUpperCase() + strings[1];
}

const name = 'aqua';
console.log(shout`こんにちは、${name}です！`);
// → こんにちは、AQUAです！
```
- shout 関数が呼ばれる
- strings → テンプレートの文字列部分（配列）
- values → ${} の中の値たち（配列）


## 使う場面
- HTMLのサニタイズ（XSS対策）
- 多言語対応（i18n）
- カスタムフォーマット（SQLクエリ、CSSなど）
- スタイルライブラリ（styled-components）でも使われてる

## 基本の書き方
```js
function myTag(strings, ...values) {
  console.log(strings); // ["Hello, ", "!"]
  console.log(values);  // ["Alice"]
  return strings[0] + values[0].toUpperCase() + strings[1];
}

const name = "Alice";
const result = myTag`Hello, ${name}!`;
console.log(result); // "Hello, ALICE!"
```
- 最初の引数 strings → テンプレートの文字列部分の配列
- 残りの引数 values → ${} に入った値
- 自由に加工して文字列を返すことができる
