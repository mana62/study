# オブジェクト
### オブジェクトのキーについて
- オブジェクトの中では、予約語（letなど）が使える
- オブジェクトのキーは全て**文字列**として扱われる
- オブジェクトの中で定義された関数＝メソッド
- オブジェクトの中で定義された値＝プロパティ
- オブジェクトのキーは文字列として定義することができる

```js
// 例：文字列として定義することができる
// 実質「name」も文字列として変換される
const person = {
    name : '太郎',
    '-3' : 3,
    'space ok' : 'OK',
    '3' : 3,
}
```

- [] 角かっこで式をキーとして扱うことができる

```js
const like = 'like'; // 変数を定義
// 例：角かっこで式をキーとして扱うことができる
// 実質「name」も文字列として変換される
const person = {
    name : '太郎',
    '-3' : 3,
    'space ok' : 'OK',
    '3' : 3,
    ['like'] : ['animals', 'ice cream'], // 外にある変数を使うこともできる
}
```

### 呼び出し方
- オブジェクトの呼び出し方は２種類ある
①ドットを使う
②角かっこを使う

角かっこを使うことで、式・数字なども呼び出し可能
```js
console.log(person['space ok']); // OKが出力される
```

### 取得方法の種類
①`for in` → オブジェクトの場合は`for of`ではなく`for in`（ただし 配列には使わない方がいい（順番が保証されないため））
②`Object.keys` → キーのみ取得
③`Object.values` → 値のみ取得
④`Object.entries` : オブジェクト → 配列（entries）に変換する
⑤`Object.fromEntries` : → 配列（entries） → オブジェクトに変換する

### オブジェクトの編集
```js
person.name = '新しい名前';（基本的にドット+プロパティ / 角かっこでもOK）
```

### オブジェクトの追加
```js
person.newObj = '新しいオブジェクト';（基本的にドット+プロパティ / 角かっこでもOK）
```

### オブジェクトの削除
```js
delete person.name; （delete演算子を使う / 角かっこでもOK）
```

### `Object.assign`について
- オブジェクトとオブジェクトを合体させるメソッド
- スプレッド構文との違いは、新しいものを生成するのではなく既存のオブジェクトを使う
  - `Object.assign(target, source)` → target が書き換わる
  - `{ ...obj1, ...obj2 }` → 新しいオブジェクトを返す

### オブジェクトのメソッドの省略記法
❌ 省略記法なし
```js
const obj = {
  hello: function () {
    console.log("こんにちは")
  }
}
```

✅ 省略記法あり
```js
const obj = {
  hello() { // : functionが不要
    console.log("こんにちは")
  }
}
```

### 省略記法のメリット
① コードが短くて読みやすい
② クラスのメソッドと同じ書き方になる
③ 内部的に「メソッド」として扱われる

### console
- `console.log`より`console.dir`の方がいい（objの中身を階層的にみれる）

## ⚠️注意
```js
{ a: 1 }
```
これが 文の先頭に来ると、
- オブジェクト ❌
- ブロック文 ⭕️
として解釈される


NG：`{ a } = obj;`
OK：`({ a } = obj);`

必ず()で囲む！！
「オブジェクトの分割代入だよ」と明示できる


下記の場合はOK、問題なし（先頭じゃないから）
```js
const { a } = obj;
let { b } = obj;
```

→ {} が文頭に来たら、ブロックだと思われる
→ 先頭に{}が来る時は()で囲む
