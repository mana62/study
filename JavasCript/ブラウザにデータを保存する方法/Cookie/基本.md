# Cookieの基本

## Cookieの基本形
**JavaScriptで作る**
```js
// 定型文
document.cookie = "name=value"
```

例:
```js
document.cookie = "user=tanaka"
```

## 属性をつける
**基本形に 属性を追加するだけ**
```js
// 定型文
document.cookie = "name=value; 属性"
```

例:
```js
document.cookie = "user=tanaka; Secure"
```

## よく使う書き方パターン
**有効期限**
```js
document.cookie = "user=tanaka; max-age=3600"
```
- 1時間保存

**パス指定**
```js
document.cookie = "user=tanaka; path=/"
```
- サイト全体で使える

**Secure**
```js
document.cookie = "user=tanaka; Secure"
```
- HTTPSのみ

**SameSite**
```js
document.cookie = "user=tanaka; SameSite=Lax"
```
- CSRF対策

## 実務でよく見る形
```js
document.cookie = "session=abc123; path=/; Secure; SameSite=Lax"
```

## サーバーの場合
サーバーでは
```js
Set-Cookie: session=abc123; Path=/; Secure; HttpOnly; SameSite=Lax
```
👉 これも 同じ構造

## まとめ
Cookieは
```js
name=value; 属性; 属性; 属性
```
この形だけ

⚠️ 実務では
- Cookie
- localStorage
- sessionStorage
を使い分ける

Cookieは主に
- ログイン認証（セッション）
で使われる
