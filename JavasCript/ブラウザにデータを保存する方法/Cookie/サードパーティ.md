# サードパーティCookie

## パーティ（party）の意味
- Webでは サイトの関係性を表す

| 用語        | 意味       |
| --------- | -------- |
| ファーストパーティ | 今見ているサイト |
| サードパーティ   | 別のサイト    |

例:
今見ているサイト
```bash
amazon.com
```
この時ファーストパーティ＝同じサイト
```bash
amazon.com
```
**サードパーティ**＝別サイト
```bash
google.com
facebook.com
ads.com
```

## サードパーティCookie

例:
今見ているサイト
```bash
news.com
```
そのページの中に広告があり
```bash
ads.com
```
のコードが読み込まれる
すると
```bash
ads.com のCookie
```
が保存されることがある
👉 これが **サードパーティCookie**


```bash
あなた
 ↓
news.com (今見ている)
 ↓
広告
 ↓
ads.com
```

この時
```bash
ads.com Cookie
```
＝ **サードパーティCookie**

## SameSiteとの関係
SameSiteは
**サードパーティCookieを制御する仕組み**

例:
`SameSite=Strict`
ならサードパーティCookie送らない

## まとめ
| 用語            | 意味          |
| ------------- | ----------- |
| ファーストパーティ     | 今見てるサイト     |
| サードパーティ       | 別のサイト       |
| サードパーティCookie | 別サイトのCookie |
