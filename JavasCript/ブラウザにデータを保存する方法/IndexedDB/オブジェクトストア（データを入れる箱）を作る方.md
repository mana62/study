# オブジェクトストア（データを入れる箱）を作る方

## オブジェクトストアとは
👉 **データを保存する箱** = **オブジェクトストア**

イメージ👇
```bash
Database (shop)
   ├ books   ← 本データを保存
   └ games   ← ゲームデータを保存
```
- Database = データベース
- **Object Store** = **テーブルのようなもの**

## 作り方
```js
db.createObjectStore("データベース名")

// 例
db.createObjectStore("books")
db.createObjectStore("games")
```

## ⚠️ 注意
- オブジェクトストアは **`upgradeneeded` の中でしか作れない**

```js
// onupgradeneeded: 初期設定
request.onupgradeneeded = (event) => {
  let db = event.target.result;

  db.createObjectStore("books");
  db.createObjectStore("games");
};
```
👉 データベースを作るタイミングでしか **構造変更できないため**

## 削除方法
```js
db.deleteObjectStore("テーブル名")

// 例
db.deleteObjectStore("games")
```
👉ただしこれも **upgradeneededの中だけ**

## 存在チェック
```js
db.objectStoreNames.contains("テーブル名")

// 例
db.objectStoreNames.contains("books")
```
👉 結果は true か false

## まとめ
```bash
Database
   ↓
Object Store（データ保存箱）
   ↓
データ
```
- `createObjectStore()` → 箱を作る
- `deleteObjectStore()` → 箱を消す
👉 作れるのは **upgradeneeded の中だけ**
- `objectStoreNames()` → 存在チェック
