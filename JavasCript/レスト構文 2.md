# レスト構文

## 配列の分割代入 × レスト構文の例
① 基本的な使い方

```js
const colors = ["red", "blue", "green", "yellow"];

const [first, second, ...others] = colors;

console.log(first);   // "red"
console.log(second);  // "blue"
console.log(others);  // ["green", "yellow"]
```
const [first, second, ...others] = colors;
分割代入で、`first, second` 1個目と2個目を指定、残りは`...others`とすること、残りの値を、`others`という変数に入れることができる

## 関数の引数で使うレストパラメー
```js
function sum(...nums) {
  return nums.reduce((a, b) => a + b, 0);
}

console.log(sum(1, 2, 3, 4)); // 10
```
`...nums` がレスト構文で、受け取った引数をまとめて配列にしてくれる

### 可変長引数との違い

```js
// レストパラメータ（可変長引数）
function sum(...nums) {
  // これはレストパラメータ
  // → 引数をまとめて配列にする
}
```
```js
// レスト構文
const [a, ...rest] = [1, 2, 3];
// これはレスト構文
// → 配列の残りをまとめる
```

## オブジェクトの分割代入でも使える
```js
const user = {
  name: "Taro",
  age: 20,
  country: "Japan",
  hobby: "games"
};

const { name, ...rest } = user;

console.log(name); // "Taro"
console.log(rest); // { age: 20, country: "Japan", hobby: "games" }
```

## 配列の途中でレスト構文を使う
```js
const nums = [1, 2, 3, 4, 5];

const [head, ...middle] = nums;

console.log(head);   // 1
console.log(middle); // [2, 3, 4, 5]
```
