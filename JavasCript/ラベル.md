# ラベル文
＝JavaScript の ラベル文（label statement） は、**文に名前（ラベル）を付ける仕組み**
＝主に **break や continue** と組み合わせて、**通常では抜けられないループやブロックから抜けるため**に使われる

## ラベル文の基本形
```js
labelName: statement
```
- labelName は好きな名前で OK
- statement にはループやブロック { ... } が入る

## 何ができるのか
### ① ネストしたループを一気に抜ける
```js
outer: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    if (j === 1) break outer; // outer ループを抜ける
  }
}
```
＝通常の break では内側のループしか抜けられないが、ラベルを使うと 外側のループを直接抜けられる

### ② continue で外側のループにスキップ
```js
outer: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    if (j === 1) continue outer; // outer の次の i へ
  }
}
```
＝通常の break では内側のループしかスキップできないが、ラベルを使うと 外側のループをスキップできる

## ⚠️ 注意点
- 使いすぎるとコードが読みにくくなる
- 一般的には 避けるべき とされる（構造化プログラミングの観点）
- どうしても必要なときだけ使うのがベスト

## まとめ
「ラベル文は、ループやブロックに名前を付けて、break/continue でそこへジャンプできる仕組み」
