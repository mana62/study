# メソッド覚え方

### ① 追加したい
```js
array.push(value);
```

### ② すでにあるか確認したい
```js
if (array.includes(value)) {
  // 重複
}
```

### ③ 全部表示したい
```js
array.forEach(item => {
  console.log(item);
});
```



① 追加

array.indexOf(value);

④ 削除
