# BigInt（ビッグインテ）
＝BigIntはプリミティブ値（標準ビルトインオブジェクトに含まれる）
＝**「めちゃくちゃ大きい整数」を扱うための新しい型**（ES2020）
＝**number型では安全に扱える整数に限界がある**
＝**その限界を超える整数を正しく計算できるのが BigInt**

```js
const n = 9007199254740992n;
```

## ①BigIntの作り方は2つ
### ● 末尾に n をつける
```js
123n
```

### ● BigInt関数で作る
```js
BigInt(123)
BigInt("123")
```

## ②BigIntは「整数専用」
＝**小数は使えない**

```js
1.1n // エラー
```

## ③⚠️ number型と混ぜて計算できない
```js
1n + 1 // エラー
```
＝**どちらかに揃える必要がある**

```js
1n + BigInt(1)
Number(1n) + 1
```

## ④BigIntは計算できるが割り算は切り捨て
```js
5n / 2n // 2n（小数にならない）
```

## ⑤比較（==, <, >）はnumberと混ぜてもOK
```js
1n < 2 // true
1n == 1 // true
```

## ⑥ Mathオブジェクトとは一緒に使えない
```js
Math.max(1n, 2n) // エラー
```

## まとめ
- 実際の用途は少なめ
- 普段は number で十分
- 超巨大な整数が必要なときだけ BigIntを使う
- 例：暗号、巨大ID、桁数の多い計算など
- BigIntは「numberでは正確に扱えない巨大な整数」を安全に扱うための型
- ただし整数専用で、numberと混ぜるとエラーになるので注意

| 項目          | 内容            |
| ----------- | ------------- |
| BigIntは新しい？ | はい           |
| 登場した年       | ES2020（2020年） |
| 目的          | 巨大な整数を正確に扱うため |
| 普段使う？       | 普通はnumberで十分  |
