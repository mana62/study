# Dateオブジェクト
＝Dateは **「日付っぽく見えるけど中身は数字」**

Dateオブジェクトは内部で下記のような数字を持っている

## よく使うもの
```js
// 現在の日時を取得（Dateオブジェクトを作成）
const d = new Date();

// -----------------------------
// 年月日を取得する
// -----------------------------

d.getFullYear();     // 年を取得（例：2026）

d.getMonth() + 1;    // 月を取得（※0〜11なので +1 が必要）
                     // 例：0=1月、8=9月

d.getDate();         // 日を取得（1〜31）

// -----------------------------
// 時刻（時間・分）を取得する
// -----------------------------

d.getHours();        // 時を取得（0〜23）

d.getMinutes();      // 分を取得（0〜59）

// -----------------------------
// 現在のタイムスタンプを取得する
// -----------------------------

Date.now();          // 現在時刻のタイムスタンプ（ミリ秒）

// -----------------------------
// ISO形式の文字列に変換する（APIでよく使う）
// -----------------------------

d.toISOString();     // "2026-02-02T10:30:00.000Z" のような形式

// -----------------------------
// 日付を変更する（例：明日にする）
// -----------------------------

d.setDate(d.getDate() + 1); // 今日の日付 +1 → 明日になる
```

```js
// 1970年1月1日から何ミリ秒経ったか

const d = new Date();
d.getTime(); // タイムスタンプ（数字）
```
＝これを**タイムスタンプ**という

## Dateの基本
### ① new Date() は「今の日時」を作る
```js
const d = new Date();
```
→ 現在時刻のDateができる

### ② Dateの中身はタイムスタンプ
```js
d.getTime();
```
→ 数字（ミリ秒）が取れる
世界共通の時間なのでタイムゾーンは関係ない

### ③ 年月日を取り出すメソッド
```js
d.getFullYear(); // 年
d.getMonth();    // 月（0〜11）
d.getDate();     // 日（1〜31）
```
⚠️ 月だけ0スタートなので注意（+1を必ずする）
例：9月 → 8

### ④ 時刻を取り出す
```js
d.getHours();      // 時
d.getMinutes();    // 分
d.getSeconds();    // 秒
```

### ⑤ UTC版もある（イギリス基準）
```js
d.getUTCFullYear();
d.getUTCMonth();
```

### ⑥ Dateは作る方法が3つある
#### ● 今
```js
new Date()
```

#### ● タイムスタンプで指定
```js
new Date(1690000000000)
```

#### ● 文字列で指定（ISO形式）
```js
new Date("2020-12-25T10:30:00Z")
```

#### ● 数字で指定（引数で順番に）
```js
new Date(2020, 11, 25, 10, 30)
```

⚠️ 月は0〜11

### ⑦ Dateは後から変更できる
```js
d.setFullYear(2025);
d.setDate(d.getDate() + 1); // 明日
```
日付を超えると自動で月が繰り上がる

### ⑧ Date同士は引き算できる（差がミリ秒で出る）
```js
const a = new Date();
const b = new Date();
```

b - a; // 経過時間（ミリ秒）

### ⑨ よく使う静的メソッド
#### ● 今のタイムスタンプだけ欲しい
```js
Date.now();
```

#### ● 文字列をタイムスタンプに変換
```js
Date.parse("2020-12-25T10:30:00Z");
```

## まとめ
| 覚えること     | 内容                                         |
| --------- | ------------------------------------------ |
| Dateの中身   | タイムスタンプ（数字）                                |
| 今を作る      | `new Date()`                               |
| 年月日を取る    | `getFullYear()`, `getMonth()`, `getDate()` |
| 月だけ0スタート  | 0=1月、11=12月                                |
| 差を計算できる   | `date2 - date1`                            |
| 今の時間だけ欲しい | `Date.now()`                               |


## よく使う
```js
const d = new Date();

d.getFullYear();
d.getMonth() + 1;
d.getDate();

Date.now();
```

## setとgetの違い
- get = 値を取得するだけ
- set = 値を変更する

⚠️ ただし実務では
- setはオブジェクトを直接変える（破壊的）
- 日付の繰り上がりが自動で起こる
