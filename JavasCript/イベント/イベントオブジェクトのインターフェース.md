# イベントオブジェクトのインターフェース

| イベントの種類       | インターフェース       | 主なプロパティ・メソッド例                         |
|----------------------|-------------------------|----------------------------------------------------|
| 一般的なイベント     | `Event`                 | `type`, `target`, `currentTarget`, `preventDefault()` |
| マウスイベント       | `MouseEvent`            | `clientX`, `clientY`, `button`, `altKey`           |
| キーボードイベント   | `KeyboardEvent`         | `key`, `code`, `keyCode`, `ctrlKey`                |
| フォーカスイベント   | `FocusEvent`            | `relatedTarget`                                    |
| フォームイベント     | `Event`                 | `submit`, `change`, `input`（共通のEvent）         |
| 入力イベント         | `InputEvent`            | `data`, `inputType`, `isComposing`                 |
| タッチイベント       | `TouchEvent`            | `touches`, `targetTouches`, `changedTouches`       |
| ドラッグ＆ドロップ   | `DragEvent`             | `dataTransfer`                                     |
| ホイールイベント     | `WheelEvent`            | `deltaX`, `deltaY`, `deltaMode`                    |
| コンポジション入力   | `CompositionEvent`      | `data`                                             |
| UIイベント           | `UIEvent`               | `detail`, `view`                                   |
| カスタムイベント     | `CustomEvent`           | `detail`（任意のデータを含められる）               |

## 補足

- すべてのイベントオブジェクトは `Event` を継承している
- `event.preventDefault()` や `event.stopPropagation()` はどのイベントでも使える基本メソッド
- `CustomEvent` を使うと、自作のイベントにデータを持たせて発火できる
