# 2進数を扱う方法

## ① Number.toString(radix)
＝ **「数字を別の進数に変換できる」**

普通は数字は10進数で書く
```js
10
```
でもコンピュータは内部で2進数を使う

数字を2進数・16進数の文字列にできる
```js
(10).toString(2);  // "1010"  ←2進数
(10).toString(8);  // "12"    ←8進数
(10).toString(16); // "a"     ←16進数
```
「これが radix（基数）」
| radix | 意味       |
| ----- | -------- |
| 2     | 2進数      |
| 8     | 8進数      |
| 10    | 10進数（普通） |
| 16    | 16進数     |


## ② parseInt(string, radix)
＝ **「別の進数を10進数に戻せる」**

文字列として書かれた2進数を数字に戻すには
```js
parseInt("1010", 2); // 10
parseInt("12", 8);   // 10
parseInt("a", 16);   // 10
```

**parseIntは第2引数(radix)を必ず書く**
```js
parseInt("08", 10); // 第二引数に明示的に10進数であることを書く
```

小数はparseIntでは変換できない
```js
parseInt("10.5"); // 10 ←小数部分消える
```

## 進数とは
「何で桁が繰り上がるか」
| 進数   | 桁が上がるタイミング |
| ---- | ---------- |
| 10進数 | 9の次で繰り上がる  |
| 2進数  | 1の次で繰り上がる  |
| 16進数 | 15の次で繰り上がる |

2進数は 0と1しか使えない
```js
10進数: 10
2進数 : 1010
```

16進数は 0〜9 + a〜f
```js
10進数: 10
16進数: a
```

## まとめ
toString() は 10進数 → 別の進数
parseInt() は 別の進数 → 10進数

## その他
2進数・8進数・16進数はこうも書ける
```js
0b1010 // 2進数 = 10
0o12   // 8進数 = 10
0xA    // 16進数 = 10
```

e記法（指数）
```js
1e5  // 100000
1e-3 // 0.001
```

数字に _ を入れて見やすくできる
```js
1_000_000 // 1000000
```
