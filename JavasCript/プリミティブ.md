# プリミティブ
＝「これ以上分解できない、超シンプルな値」

## プリミティブとは？
JavaScript には 7 種類のプリミティブがある：
①string（文字列）
②number（数値）
③boolean（true / false）
④null
⑤undefined
⑥symbol
⑦bigint
＝これらは そのままの値 で、オブジェクトみたいに中にプロパティを持たない

### プリミティブ
→ ただの値
→ 触っても中身が変わらない
→ 軽い・シンプル

例：
```js
let x = 10;       // number
let y = "hello";  // string
let z = true;     // boolean
```

### オブジェクト
→ 入れ物（箱）
→ 中にプロパティやメソッドを持てる
→ 複雑なデータ構造を作れる

例：
```js
let obj = { name: "Taro" };
let arr = [1, 2, 3];
let date = new Date();
```

## まとめ
- プリミティブ＝ただの値
- オブジェクト＝プロパティを持つ入れ物
- getter/setter はプロパティに対して動くので、実質オブジェクト専用


| 種類         | 例                 | 特徴                                   |
|--------------|--------------------|----------------------------------------|
| プリミティブ | "abc", 10, true    | 値そのもの。変更不可（immutable）。プロパティなし。 |
| オブジェクト | {}, [], new Date() | プロパティを持つ。中身を変更できる。             |
