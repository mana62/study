# 破壊的メソッド
＝元の配列を書き換えて、その配列を返すこと

## fill
＝第一引数にとった値に全てを書き換える

```js
// 定型文
array.fill(値, 開始位置, 終了位置);

// 全部埋める
[1, 2, 3].fill(9);
// → [9, 9, 9]

// 部分的に埋める
[1, 2, 3, 4].fill(0, 1, 3);
// → [1, 0, 0, 4]
```

＝fill は破壊的（元の配列を変える）
＝定型文： array.fill(値, 開始, 終了)
＝開始と終了は省略できる

## copyWithin()
＝部分コピーして上書きするだけのメソッド

```js
// 定型文
array.copyWithin(コピー先, コピー元の開始, コピー元の終了);

// コピー先だけ指定
const arr = [1, 2, 3, 4];
arr.copyWithin(1);
console.log(arr); // [1, 1, 2, 3]

// コピー元の範囲を指定
const arr = [1, 2, 3, 4, 5];
arr.copyWithin(0, 3, 5);
console.log(arr); // [4, 5, 3, 4, 5]
// コピー先：0 コピー元：3〜5（4,5）
```

＝copyWithin → 破壊的（元の配列を上書きする）
＝定型文：array.copyWithin(コピー先, 開始, 終了)
＝部分コピーして上書きするだけのメソッド
