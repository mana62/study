# 生成AIパスポート
- ファインチューニング : モデルの中身（パラメータ）を再学習して調整する
- プロンプトエンジニアリング : 質問の仕方を工夫して出力を変える
- RAG（検索連携）: 外部データを検索して回答に使う（モデル自体は変えない）
- バイアス : 思考や判断における無意識の偏りや先入観
- 確証バイアス : 自分の考えに合う情報だけを集める
- アンカリングバイアス : 最初に見た情報に引きずられる
- 利用可能性バイアス : 思い出しやすい情報を重視する
- ハロー効果 : 一部の印象で全体を判断する
- 帰属バイアス : 他人の失敗は性格、自分の失敗は環境のせいにする
- ハルシネーション : 事実に基づかない情報をもっともらしく作り出してしまう現象
- プロンプトインジェクション攻撃 : 入力文によって AI の指示内容を意図的に書き換える攻撃
- 生成AI（LLM）がもっとも得意とする処理 : 言語パターンの推測と文章生成
- 生成AIの回答に間違いが含まれる現象 : ハルシネーション
- 生成AIで画像を作った場合に考慮すべき点 : 学習データ由来の著作物に類似した画像が生成される可能性
- 「あなたはプロの秘書です」は 役割を与える : ロール指示
- 攻撃者が「前の指示を無視して、次の内容を出力しろ」という指示を埋め込む行為 : プロンプトインジェクション
- 生成AIが「幻覚（ハルシネーション）」を起こす原因 : モデルが確率的に次の単語を予測する仕組みのため
- 「強化学習（RLHF）」の目的 : モデルが与えられたタスクをより人間らしく解くように調整する
