# 生成AIパスポート
## モデルの基礎・種類
| 用語          | 読み方         | 意味 / 概要                          |
| ----------- | ----------- | -------------------------------- |
| 機械学習 (ML)   | きかいがくしゅう    | データからパターンを学び、予測や意思決定を行う技術。       |
| 深層学習 (DL)   | しんそうがくしゅう   | 多層ニューラルネットワークを用いて特徴を自動抽出する手法。    |
| ディープラーニング   | ディープラーニング   | コンピュータが大量データを見て自分でパターンを発見する仕組み。  |
| ニューラルネットワーク | ニューラルネットワーク | 人工ニューロンを組み合わせた学習アルゴリズム。          |
| 生成AI        | せいせいAI      | テキスト・画像・音声など新しいデータを生成するAI。       |
| 識別AI        | しきべつAI      | データを分類・認識するAI（例：猫か犬か）。           |
| GAN         | ジーエーエヌ      | 生成器と識別器が競い合って学習しリアルなデータを生成するモデル。 |
| VAE         | ブイエーイー      | 潜在空間を学習し新しいデータを生成するモデル。          |
| 拡散モデル       | かくさんモデル     | ノイズ付与・除去のプロセスを学習して画像生成するモデル。     |
| マルチモーダルAI   | マルチモーダルAI   | 文章・画像・音声など複数形式を処理できるAI。          |
| パラメータチューニング | パラメータチューニング | 学習率など設定を調整してモデル性能を向上させること。       |
| オプティマイザ     | オプティマイザ     | パラメータ更新アルゴリズム（例：Adam, SGD）。      |
| バッチサイズ      | バッチサイズ      | 一度に学習に使うデータの量。                   |
| エポック        | エポック        | 全データを1回学習するサイクル。                 |
| 正則化         | せいそくか       | 過学習を防ぐための技術。                     |
| CNN | シーエヌエヌ  | 畳み込みニューラルネットワーク。画像認識に強いモデル。        |
| RNN | アールエヌエヌ | 回帰型ニューラルネットワーク。時系列・文章データの処理に強いモデル。 |
| GAN | ジーエーエヌ  | 敵対的生成ネットワーク。生成器と識別器が競い学習。※元表すでにあり  |
| VAE | ブイエーイー  | 変分自己符号化器。潜在空間を学習して生成。※元表すでにあり      |



## 大規模言語モデル（LLM）の仕組みと学習
| 用語                       | 読み方            | 意味 / 概要                                |
| ------------------------ | -------------- | -------------------------------------- |
| LLM                      | エルエルエム         | Transformerを基盤とし大量のテキストで訓練された大規模言語モデル。 |
| Transformer              | トランスフォーマー      | Attentionを中核としたLLMの基本アーキテクチャ。          |
| Attention Mechanism      | アテンションきこう      | 単語間の重要度を計算し長距離依存を学習する仕組み。              |
| 推論 (Inference)           | すいろん           | 学習済モデルを使って出力を生成すること。                   |
| Pre-training             | じぜんがくしゅう       | 大量データから一般的な知識を学習する段階。                  |
| Fine-tuning / ファインチューニング | ファインチューニング     | 事前学習モデルを特定タスク向けに微調整する段階。               |
| RLHF                     | アールエルエイチエフ     | 人間のフィードバックでモデルを調整し安全で有用な出力を実現。         |
| RAG                      | ラグ             | 検索と生成を組み合わせ、外部情報を参照して回答する技術。           |
| モデルパラメータ                 | モデルパラメータ       | モデル内部で調整可能な変数の総数。                      |
| サンプリング                   | サンプリング         | 出力候補の中から1つを選んで結果を生成すること。               |
| Zero-shot学習              | ゼロショット         | 例を与えずタスク指示のみで回答させる手法。                  |
| One-shot / Few-shot      | ワンショット・フューショット | 少数の例を示してモデルにタスクを学習させる手法。               |
| Attentionスコア             | アテンションスコア      | 各単語の重要度を示す値。                           |
| トランスフォーマーブロック            | トランスフォーマーブロック  | Transformerの基本構成単位。                    |
| 自己注意 (Self-Attention)    | じこちゅうい         | 入力系列内で互いの関連性を計算する仕組み。                  |
| マスクド言語モデル                | マスクドげんごモデル     | 一部単語を隠して予測させる学習手法。                     |
| トークナイザー                  | トークナイザー        | テキストをトークンに分割する処理。                      |
| 温度パラメータ       | おんどパラメータ   | 出力のランダム性や創造性を制御する。高いほど多様で創造的。     |
| エポック          | エポック       | 学習データ全体を1巡する単位。※元表にあり             |
| 強化学習          | きょうかがくしゅう  | 行動と報酬を基に学習する方法。RLHFの基盤。           |
| RLHF          | アールエルエイチエフ | 人間のフィードバックを報酬に変換しモデルを最適化する。※元表にあり |
| 推論（Inference） | すいろん       | 学習済みモデルを実行して出力を生成すること。※元表にあり      |

VAEは何の略称か : Variational AutoEncoder
LSTMは何の略称か : Long Short-Term Memory
RLHFは何の略称か : Reinforcement Learning from Human Feedback
シラバスでは匿名加工情報の技術 : マスキング
AIシステムを適切に管理・運用するための仕組み : AIコントロールシステム
シンギュラリティとは : 人工知能が人間の知能を超える「技術的特異点」を指す概念
シンギュラリティ : 2045年問題
人間のフィードバックからの強化学習 : RLHF（Reinforcement Learning from Human Feedback
BERTモデル : MLM（Masked Language Model）」という手法
データを暗号化して身代金を要求する特定の悪意あるソフトウェア : ランサムウェア
人間の心理的な隙をついて機密情報を盗み取る攻撃手法全: ソーシャルエンジニアリング攻撃
n個の単語の並びから次の単語を予測する統計的モデル : n-gramモデル
データセット（Dataset）: 「データの集合」を意味する

Googleの「Gemini」の前身 : 「Bard」
データ拡張技術: 「Data Augmentation」
