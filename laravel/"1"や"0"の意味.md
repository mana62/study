# PHP / Laravel における `"0"` と `"1"` の意味

---

## 値ごとの評価結果

| 値    | 型     | 真偽値としての評価 | 説明                                   |
| ----- | ------ | ------------------ | -------------------------------------- |
| `"1"` | 文字列 | `true`             | 空でない文字列なので truthy            |
| `"0"` | 文字列 | `false`            | **PHP の仕様で `"0"` は falsy になる** |
| `1`   | 数値   | `true`             | 数値の 1 は truthy                     |
| `0`   | 数値   | `false`            | 数値の 0 は falsy                      |
| `""`  | 文字列 | `false`            | 空文字列は falsy                       |

---

## 実際の挙動例

```php
if ("1") {
    echo "true\n"; // 出力される
}

if ("0") {
    echo "true\n"; // 出力されない！
}

if (1) {
    echo "true\n"; // 出力される
}

if (0) {
    echo "true\n"; // 出力されない
}
```
