# エックスサーバー

## 各用語
- サーバーパネルとは
  - エックスサーバーの 管理画面 のこと

## できること
| 項目           | できること                 |
| ------------ | --------------------- |
| **ドメイン設定**   | 独自ドメインを追加、SSL 設定      |
| **メール設定**    | メールアカウント作成、転送設定       |
| **ファイル管理**   | FTP・ファイルアップロード・バックアップ |
| **データベース設定** | MySQL データベースの作成・管理    |
| **PHP設定**    | php.ini の調整、バージョン切り替え |
| **SSH設定**    | 公開鍵認証、接続許可設定          |
| **SSL設定**    | HTTPS化、証明書の管理         |
